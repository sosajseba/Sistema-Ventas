Sistema de ventas en desarrollo, conectado con una base de datos cuyo script se encuentra en esta carpeta.
Uso wampserver y MySql Workbench para la conexión y creación de la base de datos.
El sistema hasta el momento permite añadir, modificar, eliminar y buscar clientes, productos y usuarios en la base de datos.
En la carpeta src/img hay imagenes del sistema en funcionamiento ya que para probarlo correctamente deberían crear la base de datos y conectarla. 
Los archivos .form y algunas partes de código de las vistas son generados automáticamente por la librería Swing de Java.
